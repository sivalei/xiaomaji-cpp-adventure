📖 概念：cin = 听玩家说话

💡 cin 和 cout 是一对好朋友

cout << "你叫什么名字？"; // 对玩家说话
cin >> name; // 听玩家说话，存到name里
cout << "你好，" << name; // 再对玩家说话

🎯 记住：cin 用 >>（像把东西拿进来），cout 用 <<（像把东西送出去）

🔍 逐行解读

代码蛋仔比喻
cin >> x;💬 用耳朵听，把听到的存到宝箱x里
cout << x;📢 用嘴巴说，把宝箱x里的东西拿出来展示
>> 和 <<➡️ 和 ⬅️ 箭头方向不一样哦！
