跑步练习

点击展开

"想象你围着操场跑 5 圈，" for 循环就是：for(int i=1; i<=5; i++) { cout << "第" << i << "圈" << endl; }

计算机自动帮你数圈！不需要手动写 5 遍代码！

📖 概念讲解

for 循环结构：初始化；条件；变化

for(int i = 1; i <= 5; i++) {
    cout << i << " ";
}
// 输出：1 2 3 4 5

📌 int i=1 初始化 | i<=5 条件 | i++ 每次+1

while 循环：只要条件为真，就一直重复

int i = 1;
while(i <= 5) {
    cout << i << " ";
    i++;
}
// 输出：1 2 3 4 5

📌 注意：如果忘了写 i++，就会变成死循环！

❌ 死循环：忘记更新循环变量 → 程序永远不会停

❌ 条件写反：i > 5 如果 i 从 1 开始，永远不会进入循环

✅ for vs while：知道次数用 for，不知道次数用 while
