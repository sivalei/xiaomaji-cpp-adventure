📖 概念讲解

运算符名称示例结果
+加法3 + 58

/除法10 / 33
%取余10 % 31

看看这些代码在 C++ 中怎么写：

#include <iostream>
using namespace std;

int main() {
    int a = 15, b = 4;
    
    cout << "加法: " << a + b << endl;   // 19
    cout << "减法: " << a - b << endl;   // 11
    cout << "乘法: " << a * b << endl;   // 60
    cout << "除法: " << a / b << endl;   // 3
    cout << "取余: " << a % b << endl;   // 3
    
    return 0;
}

❌ 整数除法丢掉小数：10 / 3 = 3，不是 3.333

❌ % 只能用于整数

❌ 优先级：乘除取余 > 加减，括号可以改变顺序

✅ 小技巧：(a + b) * c 先算括号里的
