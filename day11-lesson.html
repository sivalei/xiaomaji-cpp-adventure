Concepts

// Linear Search - check one by one int find(int a[], int n, int target) { for(int i=0; i<n; i++) if(a[i]==target) return i; return -1; // not found }
// Bubble Sort - swap neighbors for(int i=0; i<n-1; i++) for(int j=0; j<n-i-1; j++) if(a[j]>a[j+1]) swap(a[j], a[j+1]);

Search O(n): check every item. Sort O(n^2): compare and swap. This builds algorithmic thinking - understanding efficiency matters!
