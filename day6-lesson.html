📖 概念讲解

数组 = 一排连续的宝箱，每个有编号（下标）

// 定义长度为 5 的整型数组
int coins[5] = {10, 20, 30, 40, 50};
//      下标:    0   1   2   3   4

cout << coins[0];  // 输出 10
cout << coins[2];  // 输出 30

📌 下标从 0 开始！第一个元素是 [0]

用 for 循环遍历数组：

int arr[5] = {10, 20, 30, 40, 50};
int sum = 0;
for(int i = 0; i < 5; i++) {
    sum = sum + arr[i];
}
cout << sum;  // 150

❌ 越界：5个元素，下标 0~4，访问 arr[5] 会出错

❌ 下标从 0 开始（不是 1）—— 最常犯的错误！

✅ 数组大小必须是常量，不能是变量
