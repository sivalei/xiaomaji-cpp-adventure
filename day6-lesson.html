🎬 故事导入

小蛋仔要盖一座100层的塔！如果一行一行写代码，要写100次……

有了循环，只要3行代码就能重复100次！循环就像蛋仔的「重复录制」功能！

📖 概念讲解

// for 循环：知道次数用 for
for(int i=1; i<=10; i++){
cout << i << " ";
}

// while 循环：不知道次数用 while
int i=1;
while(i<=10){
cout << i << " ";
i++;
}

💡 for 循环像数数（1,2,3...），while 循环像「一直跑直到...」
