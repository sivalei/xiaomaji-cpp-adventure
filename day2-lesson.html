📚 Today's Concepts

int = whole nums
string = text
double = decimals
Variables
int age = 9; // whole number
string name = "Egg"; // text, needs quotes
double score = 95.5; // decimal number

cout << name << " is " << age; // output: Egg is 9
cout << "Score: " << score; // output: Score: 95.5

Think of variables like MINECRAFT chests: each has a label (name), a type (what fits), and stored contents!

int = whole numbers (1, 99, -5). string = text ("hello"). double = numbers with decimal (3.14, 95.5).
