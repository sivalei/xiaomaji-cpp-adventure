小蛋仔来到了「代码王国的道具店」！

店长说："这里有很多宝箱，可以存放你的金币、钻石、名字！"

2

"但你要先学会怎么使用这些宝箱哦～"

店长指着三种宝箱：

🧰 整数宝箱（int）→ 存金币数量（100、50、10...）
🧰 小数宝箱（double）→ 存精确坐标（3.14、2.5、0.8...）
🧰 文字宝箱（string）→ 存名字（"小彩虹"、"蛋仔派对"...）
3

"学会变量，你就能让程序记住任何东西！"

🐣 让我们开始吧！

📖 概念：变量 = 宝箱

💡 变量就像蛋仔的宝箱

int gold = 100; // 🧰 整数宝箱，存100金币
double x = 3.14; // 🧰 小数宝箱，存坐标3.14
string name = "小彩虹"; // 🧰 文字宝箱，存名字

🎯 记住：变量在使用前必须先定义！

🔍 逐行解读

代码蛋仔比喻
int gold = 100;🧰 创建一个宝箱，名字叫gold，里面放100金币
gold = 200;📝 把宝箱里的金币改成200
cout << gold;📢 把宝箱里的东西拿出来展示
