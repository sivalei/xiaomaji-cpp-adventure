🧠 数学思维：分解问题

数学应用题的思考方式

就像做数学应用题一样，我们要学会拆解问题：

题目：小明有 10 个苹果，他吃了 3 个，又买了 5 个，现在有多少个苹果？

拆解步骤：

  1. 先算出吃了之后剩下多少：10 - 3 = 7
  2. 再算上新买的：7 + 5 = 12
  3. 最终答案：12 个苹果

在 C++ 中，每一个步骤都可以写成一个函数！

💻 C++ 概念讲解：函数

函数就像一个小助手，你给它一些数据，它帮你完成特定的任务，然后返回结果。

函数的定义

// 函数定义：计算两个数之和
int add(int a, int b) {
    int result = a + b;  // 计算
    return result;       // 返回结果
}

函数的调用

int main() {
    int sum = add(3, 5);  // 调用函数，传入 3 和 5
    cout << "和是：" << sum << endl;  // 输出：8
    return 0;
}

关键概念

  • 函数名：add（给函数起个名字）
  • 参数：int a, int b（传入的数据）
  • 返回值：return result（返回的结果）
  • 返回类型：int（返回值的数据类型）
