🧠 数学思维：序列与索引

数列的概念

在数学里，我们学习过数列：a₁, a₂, a₃, ... , aₙ

每一个数都有它的位置（下标），比如：

数列：2, 4, 6, 8, 10

  • a₁ = 2（第1项）
  • a₂ = 4（第2项）
  • a₃ = 6（第3项）

在 C++ 中，数组就是这样的序列！但是注意：

💡 C++ 数组的索引从 0 开始！

数组：arr[0]=2, arr[1]=4, arr[2]=6, arr[3]=8, arr[4]=10

💻 C++ 概念讲解：数组

数组是一种可以存储多个相同类型数据的容器。

数组的定义

// 定义一个有 5 个整数的数组
int coins[5] = {10, 20, 30, 40, 50};

// 访问数组元素（使用索引）
cout << coins[0] << endl;  // 输出：10
cout << coins[2] << endl;  // 输出：30

// 修改数组元素
coins[1] = 25;  // 把第2个元素改成 25

遍历数组

// 使用 for 循环遍历数组
for (int i = 0; i < 5; i++) {
    cout << "coins[" << i << "] = " << coins[i] << endl;
}

关键概念

  • 定义：数据类型 数组名[大小]
  • 索引：从 0 开始，到 大小-1 结束
  • 访问：数组名[索引]
  • 遍历：使用 for 循环
